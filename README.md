# Frontend Basics Training

This repository contains my Week 1 projects from the Frontend Basics Training program.

##  Projects Included

### 1. Profile Page
A simple personal profile page built using HTML and CSS.

### 2. Todo App
A beginner-friendly JavaScript todo app where you can:
- Add tasks  
- Mark tasks as done  
- Delete tasks  

##  How to Run the Projects

### Profile Page
1. Open the `profile-page` folder  
2. Open `index.html` in your browser  

### Todo App
1. Open the `todo-app` folder  
2. Open `index.html` in your browser
